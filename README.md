# 学生照片采集系统

一个基于PyQt5和OpenCV开发的学生照片采集系统，支持人脸检测、智能裁切、多尺寸拍照、Excel导入等功能，可用于学校批量采集学生证件照。

## 功能特点

- 📷 **摄像头实时预览**：支持多摄像头切换，实时显示摄像头画面
- 👤 **智能人脸检测**：自动检测人脸并进行智能裁切，支持CPU/GPU加速
- 📏 **多尺寸支持**：内置1寸、小2寸、大2寸、5寸、7寸等常用证件照尺寸，支持自定义尺寸
- 📊 **Excel批量导入**：导入包含学籍号和姓名的Excel文件，批量采集学生照片
- 💾 **自动命名保存**：按学籍号自动命名并保存照片，支持自定义保存路径
- ⚡ **GPU加速**：支持CUDA加速的人脸检测，提升处理速度
- 🔄 **便捷操作**：支持上一人/下一人切换、重拍等操作

## 环境要求

### 系统要求
- Windows 10/11 (推荐)
- Python 3.7+

### 依赖库
```bash
pip install opencv-python PyQt5 pandas numpy pywin32 wmi
```

> 提示：如果需要GPU加速，需安装支持CUDA的OpenCV版本和对应的CUDA工具包

## 安装使用

### 1. 克隆仓库
```bash
git clone https://github.com/a342617810/stu_photo.git
cd stu_photo
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 准备人脸检测器文件
将`haarcascade_frontalface_default.xml`文件放在程序目录下（可选，程序会自动尝试从OpenCV路径加载）

### 4. 运行程序
```bash
python student_photo_capture.py
```

## 使用指南

### 基本操作流程
1. **导入Excel文件**：点击"导入Excel"按钮，选择包含"学籍号"和"姓名"列的Excel文件
2. **选择保存路径**：点击"选择路径"按钮，设置照片保存目录
3. **选择摄像头**：从下拉菜单选择要使用的摄像头
4. **选择照片尺寸**：选择所需的证件照尺寸（或自定义尺寸）
5. **拍照采集**：
   - 确保学生面部出现在画面中，程序会自动检测人脸
   - 点击"拍照"按钮完成拍摄，照片会自动按学籍号命名保存
   - 如需重拍，点击"重拍"按钮
   - 使用"上一人"/"下一人"切换学生

### 高级功能
- **GPU加速**：如果系统支持CUDA，可在"计算设备"中选择GPU模式，提升人脸检测速度
- **自定义尺寸**：选择"自定义尺寸"，输入宽度和高度，点击"应用"即可使用自定义尺寸

## Excel文件格式要求
Excel文件必须包含以下两列：
| 学籍号 | 姓名 |
|--------|------|
| 001    | 张三 |
| 002    | 李四 |

支持.xlsx和.xls格式。

## 常见问题

### Q1: 无法打开摄像头
- 检查摄像头是否被其他程序占用
- 确认摄像头驱动已正确安装
- 尝试在摄像头选择下拉菜单中切换其他选项

### Q2: 人脸检测失败
- 确保光线充足，学生面部清晰可见
- 检查`haarcascade_frontalface_default.xml`文件是否存在
- 尝试切换到CPU模式（如果使用GPU模式失败）

### Q3: 保存照片失败
- 检查保存路径是否有写入权限
- 确保路径中不包含特殊字符或中文（部分系统可能存在兼容问题）
- 确认学籍号列数据格式正确

### Q4: 程序启动报错
- 检查Python版本是否为3.7+
- 确认所有依赖库已正确安装
- 检查QT_PLUGIN_PATH环境变量是否正确设置

## 技术架构

- **界面框架**：PyQt5 - 提供直观的图形用户界面
- **图像处理**：OpenCV - 负责摄像头采集、人脸检测、图像裁切和保存
- **数据处理**：Pandas - 处理Excel文件导入和学生信息管理
- **性能优化**：支持CUDA GPU加速的人脸检测
